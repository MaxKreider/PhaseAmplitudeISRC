Steps to produce Figure 4.5 (important - must complete steps in the order listed)

1. Run the script "K.m" to generate the K function for the circadian rhythm model
   - output: (a) plots of the K function
	     (b) .mat files containing the K function data (up to order 15)

2. Run the script "ResponseCurves.m" to generate the PRC and IRC for the model.
   - output: (a) plots of the PRC and IRC functions
	     (b) .mat files containing the PRC and IRC data (up to order 8)

3. Run the script "Plot.m" to produce the plot in Figure 4.5 
   - output: (a) plot in Figure 4.5 (NOTE: takes around a minute to execute the script)
